# 趋势跟进与多版本出图策略

本文件用于把当季视觉趋势转成可测试的 AI 出图版本。趋势不是照搬流行词，而是把颜色、材质、构图、氛围和人群情绪转成可控提示词。

## 近期趋势信号

以下趋势来自 2026 年前后的公开趋势报道和 AI 电商相关研究，生成前仍要结合具体平台和品类判断。

| 趋势 | 可转成的电商视觉 | 适合场景 |
| --- | --- | --- |
| AI 辅助多版本 | 同一商品按不同风格、人群和背景批量出图，再筛选 | 主图 A/B、详情页首屏、活动图 |
| 真实与专业并存 | 用 AI 提高画面质感，但保留真实规格、份量和使用场景 | 外卖、生鲜、服饰、家居 |
| 个性化风格测试 | 同一商品面向不同用户偏好生成不同候选图 | 淘宝/拼多多/抖音主图，小红书封面 |
| Neo Deco / 轻奢几何 | 几何框、金属边、珠宝色、拱形构图 | 饰品、礼盒、香氛、茶酒 |
| 冷感科技 / 冰蓝 | 冰蓝、银白、玻璃材质、虹彩、未来灯光 | 数码、小家电、美妆、潮流活动 |
| 戏剧化与华丽感 | 红丝绒、舞台光、金色、深酒红、仪式感 | 节日礼盒、酒水、香氛、活动图 |
| 复古怀旧与手作 | 胶片颗粒、老海报、纸张、手写感、木纹 | 食品、地方特产、文创、服饰 |
| 真实生活方式 | 自然光、居家/餐桌/户外场景、少硬广 | 小红书、抖音、微信小店、母婴 |
| 鲜明色彩与情绪色 | 巧克力棕、酒红、深青绿、冰川蓝、富金色、卡其、暖白 | Banner、节日图、品牌风格测试 |

## 每次出图建议生成几版

| 任务 | 建议版本 | 说明 |
| --- | ---: | --- |
| 普通主图 | 3 | 爆款促销、真实生活、高级极简 |
| 重点商品主图 | 5 | 加入 1 到 2 个潮流风格 |
| 活动图/Banner | 3 | 一个稳妥版、一个强促销版、一个趋势版 |
| 小红书/抖音封面 | 5 | 多测生活方式、潮流色、人物/场景角度 |
| 外卖菜品图 | 2 到 3 | 真实食欲为主，不建议过度潮流化 |
| Amazon 主图 | 1 到 2 | 白底主图严格合规，趋势只放副图/A+ |

## 推荐多版本结构

| 版本 | 目的 | 风格 |
| --- | --- | --- |
| A 稳妥转化版 | 保底可用，突出卖点 | 高转化爆款或真实食欲 |
| B 品质版 | 提升信任和客单价 | 高级极简、杂志大片 |
| C 场景版 | 让用户代入使用 | 真实生活感、自然产地 |
| D 潮流版 | 测年轻用户点击 | 冷感科技、Y2K、复古怀旧、异星未来感 |
| E 礼品/活动版 | 测仪式感和节日消费 | Neo Deco、戏剧歌剧风、新中式雅致 |

## 风格提示词写法

```text
Style direction:
Use "{风格名}" as the main style. Apply it through color palette, background material, lighting, composition, typography mood, and small decorative accents. The product remains dominant and accurate.

Audience:
Optimize for {目标人群}. The first glance should communicate {第一眼信息}.

Trend control:
Use trend elements subtly. Do not let them hide the product, reduce text readability, or create misleading product claims.
```

## 潮流元素转译表

| 潮流词 | 不建议直接写 | 更适合写成 |
| --- | --- | --- |
| 高级 | 高级感、很高级 | `restrained layout, premium material detail, soft studio light, limited color palette` |
| 潮流 | 很潮、网红 | `current social-media-friendly composition, bold but clean visual hook` |
| 复古 | 复古风 | `warm film tone, subtle grain, nostalgic poster mood, modern readable typography` |
| 科技 | 科技感 | `icy blue and silver palette, glassmorphism, clean futuristic lighting` |
| 新中式 | 国风 | `modern Chinese elegance, ink-wash texture, jade/wood accents, refined whitespace` |
| 轻奢 | 奢华 | `Neo Deco geometry, brass accents, deep jewel tones, elegant product staging` |
| 真实 | 真实一点 | `authentic lifestyle scene, natural daylight, accurate product scale, no over-retouching` |

## 趋势更新流程

1. 每月或每次大促前查一次平台和社媒趋势：淘宝/天猫活动页、抖音/小红书热门封面、美团/饿了么同类店铺、Pinterest/设计媒体趋势。
2. 只提炼可控元素：颜色、材质、构图、字体气质、背景和人群情绪。
3. 不直接模仿具体品牌、店铺、设计师或受版权保护的画面。
4. 每个趋势至少做一个“稳妥版本”对照，避免全押在新风格上。
5. 记录风格名称、提示词、成品路径和优缺点，沉淀到 [生成记录模板.md](生成记录模板.md)。
