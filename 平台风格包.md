# 平台风格包

用于按平台快速选择默认视觉组合。它解决的是“这个平台应该先测哪些风格”的问题。实际生成时，优先结合 [品类默认风格组合.md](品类默认风格组合.md) 和 [多风格设计库.md](多风格设计库.md) 调整。

## 使用方式

```text
按 {平台} 的 {3 版/5 版} 风格包，为 {商品} 生成主图。
商品核心卖点：{卖点}
目标尺寸：{尺寸}
每个风格单独生成，不要做拼图。
文件名标明平台、商品、风格、序号和尺寸。
```

## 默认选择规则

| 情况 | 先选 |
| --- | --- |
| 用户只说“做主图” | 平台 3 版包 |
| 用户说“高转化” | A/B/C 版本优先 |
| 用户说“多风格” | 平台 5 版包 |
| 用户说“跟潮流” | 在 5 版包里加入 D/E 潮流版 |
| 用户说“外卖菜品图” | 外卖 A/B/C，少字或无字 |
| 用户说“跨境/Amazon” | 合规白底版优先，再做副图风格 |
