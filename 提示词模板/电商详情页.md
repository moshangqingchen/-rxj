# 电商详情页提示词模板

适用于淘宝、天猫、京东、拼多多、抖音、快手、微信小店等移动详情页和 PC 详情页。建议按屏生成，不要一次生成超长整图。

## 详情页整体结构

| 屏幕 | 目标 | 常用内容 |
| --- | --- | --- |
| 01 首屏 | 立刻说明商品和最大卖点 | 主视觉、主标题、核心利益点 |
| 02 卖点 | 讲 1 个关键购买理由 | 功能、口感、材质、设计 |
| 03 细节 | 证明品质 | 微距、切面、工艺、包装 |
| 04 参数 | 降低决策成本 | 尺寸、重量、容量、规格 |
| 05 场景 | 帮用户想象使用 | 家庭、办公、出游、餐桌 |
| 06 售后 | 建立信任 | 发货、包装、保存、售后 |

## 单屏详情页模板

```text
Use case: ecommerce product detail page
Asset type: {平台} mobile detail page section, generated as one finished image with native Chinese text.

Primary request:
Create section {序号} of a mobile ecommerce detail page for {商品}. This section focuses only on: {本屏主题}. Generate the final image directly, including product visuals, layout, Chinese headline, short body copy, and icons/tags if needed.

Canvas:
{尺寸}, vertical mobile detail page section. Recommended width 750px equivalent, height 1000-1500px equivalent.

Subject:
{商品} shown clearly and accurately. Visual should support {本屏主题}. Use realistic product details and avoid exaggeration.

Layout:
Mobile-first vertical layout, large readable Chinese headline, short supporting copy, generous spacing, no tiny dense text. Keep text inside safe margins and do not overlap the product.

Text:
Render only the following Simplified Chinese text, verbatim, crisp and readable:
Headline: "{标题}"
Subheadline/body: "{正文}"
Tags/points: "{卖点1}" "{卖点2}" "{卖点3}"

Visual style:
{风格}: clean ecommerce detail page, premium but practical, consistent with {平台} and {品类}.

Constraints:
No watermark, no random characters, no QR code, no contact information, no unauthorized logo, no fake certificate, no exaggerated promise, no medical/absolute claims. Avoid {禁用内容}. Text must be readable on mobile.
```

## 首屏模板

```text
Create a high-conversion first screen for a {平台} mobile detail page for {商品}. The first screen must immediately show what the product is and why it is worth buying.

Canvas:
{尺寸}, vertical.

Content:
Large hero product image, strong headline "{主标题}", supporting line "{副标题}", 2-3 concise purchase-reason tags: "{标签1}" "{标签2}" "{标签3}".

Composition:
Hero product occupies the upper/middle visual focus. Text is large and mobile-readable. Clean background with brand/category atmosphere. No clutter.

Constraints:
No random text, no cropped words, no QR code, no phone number, no fake awards, no exaggerated guarantee.
```

## 参数屏模板

```text
Create a clean parameter/specification section for {商品}.

Canvas:
{尺寸}, vertical mobile detail section.

Text:
Render this accurate information only:
{参数列表}

Layout:
Use a simple table/card-like information layout with large readable text, product detail image or diagram-like visual, clean spacing.

Constraints:
Do not invent missing parameters. If information is unknown, leave it out instead of guessing. No fake certification marks.
```

## 详情页注意事项

1. 每屏只讲一个重点，避免把所有卖点塞在一张图里。
2. 食品、生鲜、保健、母婴、美妆等类目避免功效绝对化。
3. 参数屏不能编造尺寸、重量、检测报告、认证编号。
4. 最终上传前按平台宽度重新导出或裁切。
