# 提示词模板使用说明

本目录按图位提供 AI 出图提示词模板。使用时优先复制对应模板，再替换占位内容。

## 占位符

| 占位符 | 含义 |
| --- | --- |
| `{平台}` | 淘宝、拼多多、美团、Amazon 等 |
| `{图位}` | 主图、白底图、详情页、菜品图、Banner 等 |
| `{商品}` | 商品或菜品名称 |
| `{品类}` | 农产品、肉类、童装、饰品、饮品等 |
| `{尺寸}` | 800x800、750x1200、800x600 等 |
| `{核心卖点}` | 1 到 3 个最重要购买理由 |
| `{主标题}` | 图片里要出现的主标题 |
| `{副标题}` | 图片里要出现的副标题 |
| `{标签}` | 2 到 4 个短标签 |
| `{风格}` | 高转化爆款、高级极简、真实生活感、冷感科技、Neo Deco 等 |
| `{目标人群}` | 价格敏感、品质敏感、年轻种草、家庭采购、礼品送礼等 |
| `{平台风格包}` | 淘宝 5 版包、小红书 3 版包、外卖 5 版包等 |
| `{品类风格组合}` | 生鲜默认组合、外卖快餐组合、饰品礼品组合等 |
| `{禁用内容}` | 不允许出现的元素、文案或风格 |

## 通用要求

1. 如果用户要生图或改图，直接调用 image generation，不只输出提示词。
2. 如果图片需要文字，要求模型把文字原生生成在图片里，不做本地后期排版替代。
3. 文案要短、大、可读；主图文字越少越稳定。
4. 保持平台安全区，避免文字贴边、截断、被平台组件遮挡。
5. 生成后必须保存到本地，并告知最终路径。
6. 同一商品建议至少做 3 个差异明显的风格版本，再按缩略图识别度、文字准确率和平台适配筛选。
