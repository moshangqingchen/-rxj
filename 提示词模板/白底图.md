# 白底图提示词模板

适用于淘宝/天猫/京东/拼多多白底图、Amazon MAIN 主图、跨境平台主图、SKU 干净展示图。白底图的核心是审核友好和商品准确，不追求促销感。

## 标准白底图

```text
Use case: ecommerce catalog image
Asset type: white-background product image for {平台}.

Primary request:
Create a clean white-background product image for {商品}. The image must be suitable for {平台} {图位}. No text or promotional design.

Canvas:
{尺寸}, square 1:1, pure white background.

Subject:
Only the actual product: {商品}. Show the exact selling configuration: {规格/数量/包装}. Product must be complete, clearly recognizable, accurate in shape, color, material, and proportion.

Composition:
Product centered, fills the frame appropriately. For Amazon MAIN image, product should occupy about 85% of the image. For domestic white-background images, keep product large and clean with natural shadow only if allowed.

Lighting:
Clean studio lighting, true color, sharp details, no heavy shadow, no reflection that hides product shape.

Constraints:
No text, no logo, no watermark, no badge, no border, no props, no people, no hands, no QR code, no random labels, no barcode, no packaging words unless they are physically part of the real product packaging. No extra accessories not included in sale. Avoid {禁用内容}.
```

## SKU/规格图

```text
Use case: ecommerce SKU image
Asset type: clean variant image for {平台}.

Create a clear SKU image for {商品} variant "{规格/颜色/口味}". The variant must be immediately distinguishable from other variants.

Canvas:
{尺寸}, square 1:1, clean white or light neutral background.

Subject:
Show only this variant, accurate color/material/quantity, product centered and large. If packaging is included, show the actual packaging style without random text or fake branding.

Constraints:
No promotional text, no misleading quantity, no unrelated props, no random logo, no watermark, no distorted shape, no color shift.
```

## 白底图检查

| 项目 | 标准 |
| --- | --- |
| 背景 | 纯白或平台允许的干净浅色 |
| 文案 | 无标题、无标签、无促销 |
| 主体 | 单一售卖主体，完整露出 |
| 道具 | 不出现未售卖道具，避免误导 |
| 真实性 | 颜色、规格、数量与 SKU 一致 |
