# 多风格主图批量生成提示词模板

适用于同一个商品一次规划多套主图方向。实际调用 image generation 时，可以按风格逐张生成，确保每张图更稳定。

## 多风格规划提示词

```text
Use case: ecommerce creative direction planning
Asset type: multi-style ecommerce main image plan for {平台}.

Product:
{商品}

Platform and canvas:
{平台}, {尺寸}, {图位}

Goal:
Create {数量} visually distinct high-conversion main image directions for the same product. Each direction must keep the product accurate and dominant, but vary the visual style, audience appeal, composition, color palette, and text hierarchy.

Core selling points:
{核心卖点}

Required text:
Main headline: "{主标题}"
Subheadline: "{副标题}"
Tags: "{标签}"

Style directions:
Use {平台风格包} and {品类风格组合} as the base. Generate these directions:
1. {风格1}
2. {风格2}
3. {风格3}
4. {风格4}
5. {风格5}

For each style direction, generate a finished ecommerce main image with native Chinese text. Do not create a collage or contact sheet unless explicitly requested. Keep each version as a separate final image.

Constraints:
No watermark, no random words, no gibberish, no English filler, no QR code, no phone number, no external platform guidance, no unauthorized logo, no misleading quantity, no distorted product, no cropped text. Trend elements must not hide the product or reduce readability.
```

## 平台包调用示例

```text
Use the "淘宝 5 版平台风格包" and "农产品/生鲜默认风格组合" to create five separate main images for fresh corn. Keep the product, copy and selling points consistent, but vary style, palette, background and audience appeal.
```

```text
Use the "小红书 5 版平台风格包" and "服饰/童装默认风格组合" to create five separate cover images for a children's academy-style outfit. Avoid hard-sell poster style; make it social-media-friendly and lifestyle-oriented.
```

## 单张风格生成模板

```text
Use case: ads-marketing
Asset type: {平台} ecommerce product main image, finished image with native Chinese text.

Primary request:
Create one high-conversion main image for {商品} in the style "{风格}". The image must be directly generated as a finished ecommerce ad, including product photo, headline, selling-point tags, badge if needed, and CTA.

Canvas:
{尺寸}, square 1:1 unless specified otherwise.

Audience:
Optimize for {目标人群}. The first glance should communicate: {第一眼信息}.

Subject:
{商品} as the dominant product, accurate to the actual item and selling configuration. Product fills about 70%-85% of the image.

Style:
Use "{风格}" as the main visual direction. Apply it through palette, lighting, background material, composition and typography mood. Keep the product and text more important than style decoration.

Text:
Render only the following Simplified Chinese text, verbatim, crisp and readable:
Main headline: "{主标题}"
Subheadline: "{副标题}"
Tags: "{标签1}" "{标签2}" "{标签3}"
Badge: "{角标}"
CTA: "{CTA}"

Constraints:
No watermark, no random characters, no cropped text, no QR code, no contact information, no unauthorized logo, no misleading quantity, no distorted product. Avoid {禁用内容}.
```

## 文件命名建议

```text
{平台}_{商品}_主图_{风格名}_{序号}_{尺寸}_{日期}.png
```

示例：

```text
淘宝_农家甜玉米_主图_自然产地_01_1000x1000_20260619.png
淘宝_农家甜玉米_主图_高转化爆款_02_1000x1000_20260619.png
小红书_便携榨汁杯_封面_冷感科技_03_1000x1000_20260619.png
```
