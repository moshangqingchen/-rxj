# 多风格设计库

用于给同一商品或菜品生成多套差异明显的视觉方向。电商图不能只有一种“红黄爆款”模板；不同平台、不同人群、不同品类需要不同风格。生成时建议先选 3 到 5 个主风格，再用 [生成质量检查清单.md](生成质量检查清单.md) 筛掉文字不准、主体不清或过度追潮流的版本。

## 使用原则

1. 风格服务转化，不为了好看牺牲商品识别。
2. 一张图只用一个主风格，最多加一个轻量辅助风格。
3. 每个风格版本都要保留同一商品、同一规格、同一核心卖点，方便对比。
4. 潮流元素优先放在背景、色彩、材质、角标和氛围里，不要遮挡产品。
5. 食品和外卖图优先真实食欲感；白底图和 Amazon 主图不做潮流化装饰。

## 风格矩阵

| 风格 | 视觉特征 | 适合品类/平台 | 适合人群 | 提示词关键词 |
| --- | --- | --- | --- | --- |
| 高转化爆款 | 大主体、强标题、红黄/橙绿对比、角标、底部 CTA | 淘宝、拼多多、抖音、快手、生鲜、日用品 | 价格敏感、快速决策 | `high-conversion ecommerce ad, bold headline, clear selling tags, strong CTA, product fills 75%` |
| 高级极简 | 大留白、柔和棚拍、少字、克制标签、材质细节 | 天猫、京东、小红书、饰品、美妆、家居 | 品质敏感、送礼 | `premium minimalist, clean composition, soft studio lighting, restrained typography, luxury product photography` |
| 真实生活感 | 家庭/餐桌/使用场景、自然光、少修饰、轻文案 | 小红书、抖音、微信小店、食品、服饰、母婴 | 种草用户、家庭采购 | `authentic lifestyle scene, natural daylight, candid realistic, warm everyday setting, not over-polished` |
| 外卖真实食欲 | 45 度或俯拍、明亮自然、份量真实、无促销字 | 美团、饿了么、京东外卖 | 即时下单、午晚餐用户 | `realistic food delivery menu photo, appetizing, accurate portion, clean table, bright natural light` |
| 自然产地 | 农场、果园、木箱、自然材质、绿色/土色系 | 农产品、生鲜、水果、地方特产 | 关注新鲜和来源 | `farm-to-table, origin freshness, natural textures, harvest mood, trustworthy source` |
| 新中式雅致 | 宣纸感、玉石/木/竹、宋式留白、低饱和东方色 | 茶叶、饰品、礼品、滋补、文创 | 礼品、品质感用户 | `modern Chinese elegance, ink-wash texture, refined whitespace, warm jade and wood tones` |
| Neo Deco 轻奢 | 几何线条、金属边、扇形/拱形、深色珠宝调 | 珠宝、香氛、礼盒、酒水、高端食品 | 送礼、轻奢 | `Neo Deco, geometric frames, brass accents, jewel tones, elegant luxury, modern art deco` |
| 冷感科技 | 冰蓝、银白、玻璃/金属、渐变光、未来感 | 数码、小家电、功能型产品、跨境 | 年轻科技用户 | `icy blue, silver, glassmorphism, futuristic product lighting, clean tech visual` |
| 复古怀旧 | 胶片感、暖色颗粒、旧海报、2010s/港风/老字号 | 食品、饮品、服饰、地方品牌 | 情绪消费、年轻复古 | `retro commercial poster, nostalgic color, film grain, warm memory, modern vintage` |
| Y2K/多巴胺 | 高饱和、亮粉/电蓝/柠檬黄、贴纸感、年轻活泼 | 童装、潮玩、饮品、零食、小红书/抖音 | 年轻用户、学生 | `Y2K pop, dopamine colors, glossy stickers, playful typography, energetic layout` |
| 手作自然 | 纸张、麻布、木纹、手写感、暖白、低饱和 | 烘焙、手作、农副产品、文创 | 真实、慢生活用户 | `handcrafted, organic texture, kraft paper, linen, warm neutral, artisanal product photography` |
| 漫画潮流 | 粗描边、速度线、爆炸贴纸、强对比 | 快消、零食、年轻服饰、活动图 | 年轻冲动消费 | `comic pop layout, bold outlines, dynamic burst, street graphic, playful ad` |
| 杂志大片 | 人物/产品大片、强构图、标题像封面、克制高级 | 服饰、美妆、饰品、小红书、天猫 | 品牌感、审美用户 | `editorial magazine cover, fashion photography, strong composition, refined headline` |
| 戏剧歌剧风 | 红丝绒、舞台光、金色、深酒红、华丽帘幕 | 礼盒、节日活动、酒水、香氛 | 节日送礼、仪式感 | `opera aesthetic, velvet drapery, burgundy, candlelit luxury, theatrical composition` |
| 异星未来感 | 虹彩、冷光、球体、宇宙/外星氛围、非常规色 | 潮玩、数码、香氛、美妆、年轻活动图 | 尝鲜用户、潮流用户 | `alien-core, iridescent, opalescent, cosmic lighting, surreal futuristic atmosphere` |

## 平台风格建议

| 平台 | 推荐风格 | 谨慎使用 |
| --- | --- | --- |
| 淘宝 | 高转化爆款、自然产地、真实生活感、高级极简 | 过度抽象、文字太小 |
| 天猫 | 高级极简、Neo Deco、杂志大片、新中式雅致 | 低质感强促销贴片 |
| 拼多多 | 高转化爆款、真实生活感、自然产地 | 过度高级导致利益点不直观 |
| 京东 | 高级极简、冷感科技、清晰参数感 | 夸大功效、参数堆满首图 |
| 抖音/快手 | 高转化爆款、真实生活感、漫画潮流、复古怀旧 | 过度精修导致不真实 |
| 小红书 | 真实生活感、杂志大片、手作自然、Y2K/多巴胺 | 硬促销大字报 |
| 美团/饿了么 | 外卖真实食欲、明亮食欲、真实门店热卖 | 夸张海报化、虚假份量 |
| Amazon | 白底图、高级极简副图、清晰功能图 | 主图加文字/道具/促销 |

## 品类风格建议

| 品类 | 首选风格 | 可测试风格 |
| --- | --- | --- |
| 农产品/生鲜 | 自然产地、真实生活感、高转化爆款 | 手作自然、复古怀旧 |
| 水果 | 清爽高级、自然产地、真实生活感 | Y2K/多巴胺、杂志大片 |
| 肉类/熟食 | 真实食欲、家庭餐桌、高转化爆款 | 复古怀旧、手作自然 |
| 小家电/数码 | 冷感科技、高级极简、功能信息图 | 异星未来感、Neo Deco |
| 服饰/童装 | 真实生活感、杂志大片、Y2K/多巴胺 | 复古怀旧、新中式雅致 |
| 饰品/玉石 | 高级极简、新中式雅致、Neo Deco | 戏剧歌剧风、杂志大片 |
| 外卖菜品 | 外卖真实食欲、真实门店热卖、家庭套餐 | 年轻夜宵、明亮食欲 |

## 风格组合建议

| 目标 | 组合 |
| --- | --- |
| 提高点击率 | 高转化爆款 + 漫画潮流 + 真实生活感 |
| 提高客单价 | 高级极简 + Neo Deco + 杂志大片 |
| 做小红书种草 | 真实生活感 + 手作自然 + 杂志大片 |
| 做外卖转化 | 外卖真实食欲 + 明亮食欲 + 真实门店热卖 |
| 做节日送礼 | Neo Deco + 戏剧歌剧风 + 新中式雅致 |
| 做年轻潮流 | Y2K/多巴胺 + 异星未来感 + 复古怀旧 |
