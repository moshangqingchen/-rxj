# 文案与卖点模板

用于生成电商主图、详情页、外卖图和活动图时快速组织可用文案。文案要少、准、大，优先讲购买理由，不堆形容词。

## 主图文案结构

| 位置 | 作用 | 示例 |
| --- | --- | --- |
| 主标题 | 一眼说明商品和核心吸引点 | `农家鲜玉米`、`黑猪五花肉`、`招牌黄焖鸡` |
| 副标题 | 补充利益点 | `现摘现发 香甜爆汁`、`肥瘦相间 鲜嫩不腻` |
| 标签 | 强化购买理由 | `产地直发`、`粒粒饱满`、`冷链发货` |
| 角标/圆标 | 做视觉钩子 | `当季鲜货`、`爆款推荐`、`门店热卖` |
| CTA | 促进行动 | `放心下单`、`新鲜到家`、`今晚加餐` |

## 不同风格的文案语气

| 风格 | 文案语气 | 示例 |
| --- | --- | --- |
| 高转化爆款 | 直接、短促、利益点强 | `鲜甜爆汁`、`当季鲜货`、`放心囤` |
| 高级极简 | 克制、品质、少字 | `甄选好物`、`细节见品质`、`清爽到家` |
| 真实生活感 | 日常、亲近、像推荐 | `今天就吃它`、`一家人都爱`、`下班快手菜` |
| 小红书种草 | 轻松、场景化、体验感 | `好吃到想回购`、`拍照也好看`、`周末轻松安排` |
| 外卖真实食欲 | 即时、馋感、明确 | `热乎现做`、`下饭够香`、`午餐不纠结` |
| Neo Deco/礼品 | 仪式感、体面、精致 | `送礼有面`、`一盒见心意`、`精致好味` |
| 冷感科技 | 清晰、功能、效率 | `一键鲜榨`、`轻松清洁`、`小巧高能` |
| 复古怀旧 | 情绪、记忆、地方感 | `小时候的味道`、`老味新做`、`一口回忆` |

## 品类卖点

### 农产品/生鲜

| 方向 | 可用表达 |
| --- | --- |
| 新鲜 | `现摘现发`、`当季鲜采`、`源头直发`、`新鲜到家` |
| 品质 | `颗粒饱满`、`果肉厚实`、`严选好果`、`看得见的新鲜` |
| 口感 | `香甜软糯`、`清甜多汁`、`脆嫩爽口`、`鲜香不腻` |
| 包装配送 | `冷链发货`、`泡沫箱包装`、`坏果包赔`、`到家更安心` |

避免：`全网最低`、`全国第一`、`包治`、`降血糖`、`绝对新鲜`。

### 肉类/熟食

| 方向 | 可用表达 |
| --- | --- |
| 肉质 | `肥瘦相间`、`肉质紧实`、`鲜嫩多汁`、`纹理清晰` |
| 来源 | `源头直供`、`严选好肉`、`新鲜现切` |
| 烹饪 | `煎炒炖煮都合适`、`家常好味`、`下锅鲜香` |
| 配送 | `真空包装`、`冷链发货`、`到家更放心` |

避免：未经证明的营养功效、医疗暗示、绝对化品质承诺。

### 水果

| 方向 | 可用表达 |
| --- | --- |
| 产地 | `产地直发`、`应季采摘`、`果园鲜采` |
| 口感 | `清甜多汁`、`果肉细腻`、`甜润爽口` |
| 外观 | `果形饱满`、`色泽自然`、`严选大果` |
| 售后 | `坏果包赔`、`安心下单` |

避免：糖度、产地、等级如果没有确定信息，不要编具体数字。

### 饰品/玉石

### 外卖菜品

| 方向 | 可用表达 |

## 详情页模块文案

| 模块 | 写法 |
| --- | --- |
| 首屏 | 商品是什么 + 最大购买理由 |
| 卖点屏 | 每屏一个利益点，标题 6 到 12 字 |
| 细节屏 | 说清材质、口感、工艺、切面、包装 |
| 参数屏 | 尺寸、重量、规格、储存、食用/使用方式 |
| 场景屏 | 谁在什么场景下使用，解决什么问题 |
| 售后屏 | 发货、包装、退换、注意事项 |

## 活动图文案结构

| 层级 | 示例 |
| --- | --- |
| 活动主题 | `夏日鲜果节`、`开学焕新季`、`门店招牌上新` |
| 核心利益点 | `当季鲜采`、`多件更划算`、`新品尝鲜` |
| 行动提示 | `立即选购`、`今晚下单`、`进店看看` |
| 限制信息 | 时间、范围、规则必须真实，不要写模糊承诺 |

## 文案避雷

| 类型 | 避免表达 |
| --- | --- |
| 极限词 | `第一`、`最好`、`顶级`、`全网最低`、`永久`、`100%` |
| 医疗功效 | `治疗`、`降血糖`、`抗癌`、`根治`、`消炎` |
| 虚假承诺 | `无效退款`、`吃了就瘦`、`稳赚`、`绝不坏果` |
| 误导对比 | 无依据的前后对比、竞品贬低、虚构检测报告 |
| 平台导流 | 二维码、手机号、微信号、外部链接、跳转引导 |
