## 5. 问题与修正

| 问题 | 修正方向 |
| --- | --- |
| 文字不准/乱码 | 减少文字，改短句，要求大字清晰 |
| 主体不够突出 | 提高主体占比，弱化背景和装饰 |
| 商品/菜品不真实 | 强化真实规格、实际份量、准确结构 |
| 促销感不足 | 增加角标、标签、底部 CTA，但不堆字 |
| 平台风险 | 删除二维码、联系方式、极限词、未授权 Logo |
