# 常用出图指令示例

这些示例用于日常直接发起 AI 出图。可以复制后替换商品、平台、尺寸和卖点。

## 电商主图

```text
为我生成一套淘宝商品主图，商品是农家甜玉米，800x800，共 5 张。
方向：高转化电商主图，产品突出、促销感适中、文字清晰。
核心卖点：现摘现发、香甜软糯、颗粒饱满、坏果包赔。
图片里文字必须由 image generation 原生生成，最终图片保存到本地。
```

```text
为这个商品做 5 套不同风格的淘宝主图方案，商品是便携榨汁杯，1000x1000。
每套都要高转化，但风格要明显不同：
1. 爆款促销风
2. 高级极简风
3. 小红书生活方式风
4. 冷感科技风
5. 年轻潮流复古风
每张图都要保存到本地，并在文件名里标明风格。
```

```text
按“淘宝 5 版平台风格包 + 生鲜默认风格组合”，为农家甜玉米生成 5 张主图，1000x1000。
风格必须明显不同，但商品、规格和核心卖点保持一致。
每张图单独生成，文件名标明风格，不要做拼图。
```

```text
做一张拼多多爆款风主图，商品是黑猪五花肉，1000x1000。
要有强点击标题、3 个卖点标签和底部购买理由。
禁止出现二维码、联系方式、极限词和随机品牌。
```

## 白底图

```text
生成一张 Amazon 主图白底图，商品是便携榨汁杯，2000x2000。
只展示实际售卖主体，无文字、无 Logo、无道具、无促销元素。
商品占画面约 85%，颜色和结构真实。
```

```text
生成淘宝白底图，商品是儿童学院风套装，800x800。
纯白背景，服装完整展示，无文字、无标签、无场景道具。
```


```text

只展示我给出的参数，不要编造认证、承重或材质。
```

## 外卖菜品图

```text

真实外卖菜单图风格，45 度拍摄，有食欲但不过度滤镜。
份量与实际一致：鸡肉、土豆、青椒、米饭一份。
不要价格、满减、配送费、二维码和联系方式。
```


## 多风格测试

```text
同一个商品请不要只做一种视觉。为农家甜玉米生成 5 个主图方向：
高转化爆款、自然产地、真实家庭餐桌、清爽高级、年轻复古。
每个方向文案不要超过 1 个主标题和 3 个标签。
生成后用质量检查清单筛掉文字不准、主体不清、风格过度的版本。
```

```text
为一家外卖黄焖鸡店生成 4 套菜品图/活动图风格：
真实门店热卖、明亮食欲、年轻夜宵、家庭套餐。
不要做成虚假大促海报，份量和配菜必须符合实际售卖。
```

```text
按“小红书 5 版平台风格包 + 服饰童装默认组合”，为儿童学院风套装生成 5 张封面图。
要求有真实生活感、杂志大片、Y2K、多巴胺、复古学院风等差异。
不要硬促销，不要大字报，文字短而清楚。
```

```text
按多风格趋势化出图执行方案，为便携榨汁杯生成 5 张京东/小红书都可参考的主图。
版本结构：稳妥转化版、品质高级版、场景种草版、冷感科技版、年轻潮流版。
每张图单独生成并保存到本地，文件名标明风格。
```

## 改图

```text
把这张商品主图改成更高转化的淘宝主图。
保留商品主体，增强标题层级和卖点标签，去掉杂乱背景。
图片里的新文字必须由 image generation 原生生成，完成后保存到本地。
```

```text
把这张外卖菜品图改得更明亮、更有食欲，但不要改变份量和配菜。
不要添加价格、促销字、二维码或不存在的食材。
```
