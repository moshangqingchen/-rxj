# 我的 Codex 使用偏好

当我提出“生图”“画一张”“生成图片”“做一张图”“改图”“图片编辑”等请求时，默认直接调用 image2.0 / image generation 工具生成或编辑图片，不要只给 prompt。

处理上述 AI 生图或改图请求时，图片生成和编辑环节只调用 image2.0 / image generation；除非是为了把最终图片保存到本地、确认文件真实存在，或我明确要求，否则不要调用其他工具替代或预处理。

只有在我明确说“只要提示词”“写成 SVG”“用 HTML/CSS 实现”“做成代码资产”时，才不要调用生图模型。

当我提出“主图”“商品主图”“电商主图”“精美主图”等请求时，默认按高转化电商主图方向生成：产品突出、画面精美、卖点清晰、标题有冲击力、标签和角标增强购买理由，整体以提高点击率和转化率为目标。

主图生成不要只做普通产品摆拍，要参考常见电商商品首图/营销主图风格，结合品类加入合适的卖点文案、促销感、品牌感、平台感和干净高级的视觉排版。

当我使用 AI 生成或编辑图片时，必须把最终图片落盘保存为本地图片文件，不要只给聊天预览图。若我没有指定目录，默认在当前工作区按任务名称新建文件夹保存，并使用清晰、连续的文件名；完成后必须告诉我保存路径。
